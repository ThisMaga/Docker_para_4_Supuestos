# Página Personal

Hola, este es mi pequeño espacio web, donde podrás conocerme un poco. 

A continuación debes abrir el index.php

Este proyecto es una página web personal que presenta una biografía, una sección de proyectos y un formulario de contacto. La página permite gestionar proyectos desde una base de datos en MySQL, y está desarrollada con PHP, HTML, CSS y JavaScript, utilizando Bootstrap y jquerypara el diseño.

He tenido muchos cambios en el transcurso de la elaboración. 
Sin embargo te digo un poco en que consiste.

Página de Inicio: Presenta una breve biografía con información sobre la autora.
Sección de Proyectos: Muestra una lista de proyectos cargados desde una base de datos. Cada proyecto incluye una imagen, un título, una breve descripción y un botón "Ver más" que abre un modal con detalles adicionales.
Formulario de Contacto: Permite a los usuarios enviar un mensaje de contacto. Valida el correo electrónico antes de enviarlo.
Agregar Proyecto: Incluye un formulario para agregar nuevos proyectos a la base de datos. Al añadir un proyecto, la página se recarga automáticamente mostrando el nuevo contenido.


También en la barra superior encuentras un menú, con enlaces a las partes de la página web, y en la esquina superior derecha, encontrarás un buscador.
El cual te muestra, en la mitad de la página la busqueda.

En la creación mi página web he utilizado las siguientes tecnologías:

HTML5: Estructura de la página y contenido.
CSS3 (style): Estilos personalizados y diseño visual de la página.
Bootstrap: Un framework CSS que facilita el diseño responsivo.
jQuery: Para las interacciones dinámicas como el buscador de proyectos y la validación del formulario.
Modal de Bootstrap: Para crear ventanas modales en los proyectos.
Fuentes personalizadas: Uso de fuentes externas a través de @font-face.
